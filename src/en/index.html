Pikachu Volleyball is a classic game which was developed in Japan in 1997. The game was popular in Korea when I was young. I had played it with my friends often.

This web version is made by reverse engineering the core part of the machine code of the original game and implementing it to JavaScript. It is executed directly on the web browser so, unlike the original game, you can play it not only in Windows, but also in macOS, Linux, etc. — only if you have a keyboard. You can view the source code on GitHub.

Controls:

Tip 1:
When you select "1人でぴかちゅ~" to play against the computer, you would play as player 1 if you select by pressing the "z" key, or you would play as player 2 if you select by pressing the "enter" key.

Tip 2:
If you want to hide/show the game menu bar, press "esc" key.

Tip 3:
I've added a practice mode which is not in the original game, you can turn on this mode at "Options → Practice mode → on". I hope you like it:)

The winning score can not be set to the pts, since the score is already reached.

There's no point adjusting the winning score in practice mode:)
